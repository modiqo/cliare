# CLIARE: CLI Agent Readiness Evaluation

This folder is the long-form design packet for **CLIARE**, an independent open-source project under modiqo.

CLIARE measures how reliably agents and automation can discover, understand, safely execute, and recover while using command-line interfaces. It treats a CLI as a black-box runtime system, exercises it inside a controlled sandbox, infers a probabilistic command-shape model, and emits reproducible scorecards, issue ledgers, command indexes, persona packets, and artifact maps suitable for CI, badges, agent navigation, leaderboard submission, and long-term improvement tracking.
